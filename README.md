# UMC 203 course requiremnets

This folder is organized so each research direction can live in its own self-contained workspace instead of turning into one large script dump.

## Layout

- `direction_1_common_benchmark/`: common benchmark for the three assigned papers.
- `direction_2/`: placeholder for the second direction.
- `direction_3/`: placeholder for the third direction.
- `direction_4/`: placeholder for the fourth direction.
- `direction_5/`: placeholder for the fifth direction.

Each direction has its own code, configs, reports, and outputs.
